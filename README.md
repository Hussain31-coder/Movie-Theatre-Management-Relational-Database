Movie Theatre Management System 🎬A production-ready database management system implemented and optimized within MySQL Workbench. This project models the complete data infrastructure of a modern cinema, mapping complex operations from an abstract Entity-Relationship Diagram (ERD) into fully optimized physical tables and business-critical analytical workflows.📌 FeaturesShow & Ticketing Management: Tracks showtimes, ticket categories, pricing structures, and real-time seating availability.Operational Asset Management: Manages multiple theatre branches, individual viewing screens, seating capacities, and hardware specifications.Customer Loyalty Framework: Stores demographic profiles, automates age tracking, and links customer accounts to purchase histories.Point of Sale Integration: Unifies ticket purchases and concession snack sales under centralized transaction logs.Staff Infrastructure Management: Organizes employee roles, designations, shifts, salaries, and specific venue assignments.🗺️ Database Architecture1. Conceptual Design (ERD)The database structure relies on an Entity-Relationship Diagram that captures the synergy between different cinema sectors:Entities: Movie, Show, Ticket, Customer, Payment, Snack, Seat, Screen, Theatre, and Staff.Key Synergies: Movies link to Shows ➔ Shows generate Tickets ➔ Customers purchase Tickets & Snacks via Payments.2. Relational SchemaThe conceptual design maps directly into normalized relational tables. This design eliminates data redundancy and maintains strict data integrity across transactions.💻 Implementation & QueriesThe physical layout and business logic are fully written, deployed, and tested inside MySQL Workbench.Core Database ComponentsDDL Scripts: Structured schemas defining tables, data types, primary keys, and complex foreign key constraints.DML Scripts: Mock transactional records mimicking daily theatre activities.Business-Critical Intelligence ReportsThe repository contains an advanced analytical suite of SQL scripts engineered to answer key operational questions:Revenue Performance: Tracks total daily and monthly earnings from combined ticket sales and concession counters.Capacity Analytics: Evaluates occupancy rates to identify the most popular time slots, screen types, and movie genres.Operational Audits: Monitors transaction statuses, processing discount anomalies, and ticket cancellations.
